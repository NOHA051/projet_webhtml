À propos du Projet

Site réalisé dans le cadre du projet TI402 — Département Informatique, Efrei Paris.

Présentation du projet

Ce site vitrine a été réalisé dans le cadre du cours TI402 — Projet de Programmation Web de l'Efrei Paris (2025–2026), encadré par M. Mohamed Hamidi. L'objectif était de concevoir et développer un site web complet pour le Département Informatique de l'Efrei, en respectant une charte graphique définie, en pure HTML5, CSS3 et JavaScript.

Travail réalisé

Architecture du site

Le site comporte 8 pages HTML reliées par une navigation cohérente : une page d'accueil, une page de présentation des formations, deux sous-pages de programmes (Bachelors et Cycle Ingénieur), une page enseignants, une page vie étudiante, une page contact et la présente page « À propos ».

Fonctionnalités JavaScript

Plusieurs fonctionnalités dynamiques ont été implémentées : carrousel automatique sur la page d'accueil, système d'onglets pour les programmes, filtre de l'équipe enseignante par domaine, et validation du formulaire de contact côté client.

Charte graphique respectée

La palette de couleurs institutionnelle (Bleu Académique #0A316C, Cyan #81ECEC, Jaune #FDCB6E), les typographies Google Fonts (Montserrat Bold pour les titres, Open Sans pour le corps de texte) et les règles de mise en page (composition diagonale, triangles ▶ comme puces, logo en blanc sur fond bleu) ont été appliquées scrupuleusement sur toutes les pages.
